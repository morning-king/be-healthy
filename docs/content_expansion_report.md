## 3. 内容管理系统操作手册

目前内容管理通过代码库中的 `ContentRepository` 和 Room 数据库预填充数据进行维护。

### 3.1 如何新增内容

1.  **定位文件**: 打开 `app/src/main/java/com/behealthy/app/core/repository/ContentRepository.kt`。
2.  **找到列表**:
    - `initialQuotes`: 用于存储名言。
    - `initialPoems`: 用于存储诗词。
3.  **添加条目**:
    
    **添加名言示例**:
    ```kotlin
    QuoteEntity(
        content = "新的名言内容",
        source = "作者/出处",
        category = "分类(Motivation/Health/Life)",
        tags = "tag1,tag2"
    )
    ```

    **添加诗词示例**:
    ```kotlin
    PoemEntity(
        content = "诗词正文\n换行请用\\n",
        author = "作者",
        title = "标题",
        dynasty = "朝代(Tang/Song)",
        category = "分类",
        tags = "tag1,tag2"
    )
    ```

### 3.2 数据库更新机制
- 应用启动时，`ContentRepository` 会检查数据库是否为空。
- 如果为空（首次安装或清除数据后），会自动将 `initialQuotes` 和 `initialPoems` 写入数据库。
- **注意**: 若要更新已安装用户的数据库，需要在 `DatabaseModule.kt` 中增加新的 `Migration`，或设计同步机制（当前版本主要依靠预置数据）。

### 3.3 内容审核标准
1.  **版权**: 确保引用内容为公有领域（Public Domain）或已获授权。古典诗词和古人名言无版权风险。
2.  **准确性**: 核对原文、作者和出处，避免张冠李戴。
3.  **价值观**: 内容需积极向上，符合健康生活主题，避免消极、颓废或有争议的政治言论。
